# 简介
基于闭源AstralDice的Lua框架
# AstralDice是什么？
AstralDice是某人开发的一个可供trpg跑团骰点的骰系，并且提供了**回雪**，**JavaScript**，**Lua**语言进行扩展功能的开发
# Dream的优点
 - 快速载入Lua脚本，并且迅速生成配置文件
 - 高效快捷，实现功能只需几行
 - 提供了报错处理，让报错**不再静默**
 - 通过Termux，提供了lua与其他语言交互(如PHP)的可能
